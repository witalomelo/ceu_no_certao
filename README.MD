# Nome do Projeto

API familias do sertao

## Tecnologias Utilizadas

- Node.js
- Yarn
- Sequelize
- PostgreSQL
- bcryptjs
- jsonwebtoken

## Pré-requisitos

Antes de começar, certifique-se de que você tenha as seguintes ferramentas instaladas:

- Node.js
- Yarn

## Instruções
1. Instalar dependencias:
`yarn install`

2. Iniciar o projeto
`yarn dev`

3. Documentacao:
http://localhost:3000/api-docs
